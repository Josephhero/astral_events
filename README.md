# astral_events

This repo sends out an email with various astrological events that are happening in the following format:
**Feb 06, 2025**
**Lunar Illumination:** 61%
**Age of Moon:** 9 Days
**New Moon:** 22 Days

No Current Events
[this would show any meteor showers, visible planets, etc., if anything were happening on the current day]

**Next Equinox:** 43 Days
**Next Meteor Shower:** 43 Days
~Lyrid Meteor Shower (Apr 22/23)
